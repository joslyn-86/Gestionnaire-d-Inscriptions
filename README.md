Gestionnaire d’Inscriptions Étudiants

Application desktop développée en **C++ avec Qt** 
permettant la gestion des étudiants avec un stockage simple en fichier **CSV**.

Fonctionnalités

-  Ajouter un étudiant (Nom, Prénom, Âge, Niveau)
- Modifier les informations d’un étudiant
-  Supprimer un étudiant avec confirmation
-  Recherche dynamique en temps réel
-  Affichage sous forme de tableau (QTableWidget)
-  Sauvegarde automatique dans un fichier CSV
-  Chargement automatique au démarrage

 Technologies utilisées

- C++
- Qt
